数据库采用MYSQL请勿使用不支持的函数

MySQL Date 函数 资料查阅
  • 请使用半角数字字符和括号
  • 字符串类型例子：学校共有教职工{字段名A}人
  • 数字类型例子：{字段名A}+10*({字段名B}-{字段名C})*{字段名D}/{字段名E}
  • 日期类型例子1：TIMESTAMPADD(part,startdate,enddate)part 为year, month, day等, now()表示当前时间
  • 日期类型例子2：DATEDIFF(endData,startDate)返回两个日期之间的天数
  • 类似{字段名A}的参数可以出现在在表达式中的任意位置
  • U_GetAgeFromIdentity({字段名A})
  • U_GetBirthDayFromIdentity({字段名A})
  • U_GetDateInternal('year|month|day',{日期字段A},{日期字段B})
  • U_GetSexFromIdentity({字段名A})
  • NOW()返回当前的日期和时间
  • CURDATE()返回当前的日期
  • CURTIME()返回当前的时间
  • DATE()提取日期或日期/时间表达式的日期部分
